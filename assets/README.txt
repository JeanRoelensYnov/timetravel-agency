DOSSIER ASSETS — où déposer vos images et vidéos
==================================================

Déposez simplement vos fichiers ici avec EXACTEMENT ces noms.
Le site les détecte automatiquement. Tant qu'un fichier est absent,
un visuel dégradé élégant s'affiche à la place (le site reste joli).

IMAGES DES DESTINATIONS (vos visuels du Projet 1)
-------------------------------------------------
  assets/paris.jpg        →  carte + fiche "Paris 1889"
  assets/cretaceous.jpg   →  carte + fiche "Le Crétacé"
  assets/florence.jpg     →  carte + fiche "Florence 1504"

  Format conseillé : JPG/PNG/WebP, ~1200×1600 px (portrait) ou 1600×1200.

VIDÉO DE FOND DU HERO (optionnel)
---------------------------------
  assets/hero.mp4         →  vidéo en arrière-plan de la section d'accueil
  assets/hero-poster.svg  →  image affichée avant le chargement (déjà fournie ;
                             remplaçable par hero-poster.jpg si vous préférez)

  Si aucune vidéo n'est présente, une animation "champ d'étoiles"
  s'affiche automatiquement à la place. Aucune action requise.

CHANGER UN NOM DE FICHIER ?
---------------------------
Tout est centralisé dans js/app.js → objet DESTINATIONS
(champ "img" de chaque destination). Modifiez-y le chemin si besoin.
